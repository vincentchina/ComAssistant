# qt串口助手
在工作中经常会用到串口调试，使用了网上的某些搜索得到的串口助手，有的会经常卡死，有的还要收费。
抱着这些不满，开始编写一个属于自己的串口助手。选择QT，最重要一点，可以跨平台，在linux上也能够使用。

涉及的功能有：
- 字符串、十六进制显示切换
- 可以设置自动换行
- 可以显示发送、接收时间
- 可以自动补校验：8位校验和、16位校验和、CRC16、CRC32，同时可以指定大小端
- 可以发送自动补换行，或者自定义结束符
- 自动保存数据到文档
- 设置显示缓存大小，自定义字体大小，显示文本编码
- 可以设置自动发送
- 显示数据收发字节数
- Windows上可以扫描串口
- 提供ASCII码对照表
